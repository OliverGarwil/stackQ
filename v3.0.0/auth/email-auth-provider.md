### credential
[method]credential(email, password) returns [AuthCredentia](version /auth/auth-credential)[/method]

| Parameter |         |
| --------- | ------- |
| email  | **string** <br /> Email address. |
| password  | **string** <br /> User account password. |

#### Example

```js

    password
```
